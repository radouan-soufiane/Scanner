Security Scanner

Une application utilitaire légère développée avec Python et Tkinter pour scanner et identifier les fichiers potentiellement suspects sur votre machine.

🚀 Fonctionnalités

Scan de sécurité : Analyse les dossiers à la recherche de fichiers ayant des extensions suspectes (ex: .bat, .exe, .vbs).

Interface intuitive : Utilisation simple et rapide via une interface graphique propre.

Reporting : Affiche une liste des fichiers identifiés pour une vérification manuelle.

🛠 Prérequis

Pour exécuter ce projet, vous devez avoir Python installé sur votre machine.

📥 Installation

Cloner le projet :


Accéder au dossier :

cd security-scanner


Lancer le programme :

python security_scanner.py


📂 Structure du projet

security_scanner.py : Le fichier principal contenant la logique de scan et l'interface.

README.md : Documentation du projet.

⚠️ Avertissement

Cet outil est un simulateur éducatif. Il ne remplace en aucun cas un antivirus professionnel. Il sert uniquement à identifier des extensions de fichiers couramment utilisées dans les scripts suspects.
